## Cycle Rich Text Editor

一款基于draft.js打造的高颜值富文本编辑器，简单易用，适用于react框架。

### `npm start` 本地运行

本地开发模式<br>

打开 [http://localhost:3000](http://localhost:3000) 地址.

### 优势

- 实时热替换模块开发

- 打印详细的错误堆栈信息
